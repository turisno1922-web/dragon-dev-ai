⚡ Contoh Kode Lua Siap Pakai

Buat Part Bergerak (Tween)

BEGINNER
Part bergerak naik-turun otomatis. Taruh di Script dalam Part.
local part = script.Parent local TweenService = game:GetService("TweenService") local startPos = part.Position local endPos = startPos + Vector3.new(0, 5, 0) local info = TweenInfo.new(2, Enum.EasingStyle.Sine, Enum.EasingDirection.InOut, -1, true) local tween = TweenService:Create(part, info, {Position = endPos}) tween:Play()

GUI Button — Buka/Tutup Frame

BEGINNER
Button di layar yang membuka/menutup Frame. Taruh di LocalScript dalam ScreenGui.
local button = script.Parent.OpenButton local frame = script.Parent.MainFrame frame.Visible = false button.MouseButton1Click:Connect(function() frame.Visible = not frame.Visible end)

RemoteEvent — Kirim Data ke Server

INTERMEDIATE
Client kirim data ke Server. RemoteEvent di ReplicatedStorage.
-- LocalScript (Client) local RE = game.ReplicatedStorage.MyEvent RE:FireServer("Hello dari client!") -- Script (Server) local RE = game.ReplicatedStorage.MyEvent RE.OnServerEvent:Connect(function(player, msg) print(player.Name .. " berkata: " .. msg) end)

Leaderboard (Stats Pemain)

BEGINNER
Buat poin/gold di leaderboard. Taruh di Script di ServerScriptService.
game.Players.PlayerAdded:Connect(function(player) local stats = Instance.new("Folder") stats.Name = "leaderstats" stats.Parent = player local gold = Instance.new("IntValue") gold.Name = "Gold" gold.Value = 0 gold.Parent = stats end)

Spawn Dragon (Clone Model)

INTERMEDIATE
Spawn model dari ServerStorage ke posisi tertentu. Model harus sudah ada di ServerStorage.
local ServerStorage = game:GetService("ServerStorage") local dragonModel = ServerStorage:FindFirstChild("Dragon") local function spawnDragon(position) if dragonModel then local clone = dragonModel:Clone() clone.Parent = workspace clone:SetPrimaryPartCFrame( CFrame.new(position)) end end spawnDragon(Vector3.new(0, 10, 0))

🗺️ Roadmap Roblox Developer dari HP

Phase 1 · Minggu 1-2
Mengenal Roblox Studio
  • Install Roblox Studio di PC teman / warnet
  • Kenali interface: Explorer, Properties, Toolbox
  • Buat Part pertama & atur warna/ukuran
  • Pahami hierarchy: Workspace, ServerScriptService, dll
Phase 2 · Minggu 3-4
Belajar Dasar Lua
  • Variable, string, number, boolean
  • If/else, loop (for, while)
  • Function sederhana
  • Manipulasi Part lewat script
Phase 3 · Bulan 2
Scripting Intermediate
  • Events: Touched, ClickDetector, PlayerAdded
  • LocalScript vs ServerScript vs ModuleScript
  • RemoteEvent & RemoteFunction
  • GUI: ScreenGui, Frame, Button, TextLabel
Phase 4 · Bulan 3
Buat Game Pertama
  • Desain map sederhana (Simulator / Obby)
  • Leaderboard & DataStore (simpan data)
  • Monetisasi: GamePass & DevProducts
  • Publish game & dapatkan feedback
Phase 5 · Bulan 4+
Tingkatkan & Hasilkan Robux
  • Pelajari OOP (Object-Oriented Programming) di Lua
  • Optimasi performa game
  • Join komunitas developer Roblox Indonesia
  • Mulai jual asset / freelance dev di grup
💡 Tips dari HP tanpa PC:
Kamu bisa belajar teori, konsep Lua, dan desain game dari HP! Gunakan waktu di warnet / HP teman untuk praktek di Roblox Studio. Chat dengan Dragon Dev AI ini kapan saja untuk tanya konsep, minta contoh kode, atau minta penjelasan error.
